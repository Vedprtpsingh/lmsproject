## Program Details

### Duration
- **Standard**: 6 Months (Full-time)
- **Format**: Classroom + Lab + Project

### Batches
- **February Batch**: February to July
- **August Batch**: August to January
- **September Batch**: September to February

### Eligibility
- B.E./B.Tech. in relevant discipline
- Or M.Sc. in relevant discipline with valid GATE score

## Typical Course Structure

### Example: PGCP-AC (Advanced Computing)

**Core Subjects:**
1. Core Java (ACTS-01)
2. Advanced Java (ACTS-02)
3. Data Structures & Algorithms (ACTS-03)
4. Database Management Systems (ACTS-04)
5. Operating Systems (ACTS-05)
6. Web Technologies (ACTS-06)
7. Software Engineering (ACTS-07)
8. Advanced Web Programming (ACTS-08)

**Labs:**
- Java Programming Lab
- DBMS Lab
- Web Technologies Lab

**Project:**
- Final Project (Team-based)

### Credits
- **Theory**: 4 credits each
- **Lab**: 2 credits each
- **Project**: 6 credits
- **Total**: ~40-50 credits

## Placement Statistics

C-DAC programs typically have:
- **Placement Rate**: 80-90%
- **Average Package**: ₹4-6 LPA
- **Top Package**: ₹12-15 LPA

## Admission Process

1. **Online Application**
2. **C-CAT Exam** (C-DAC Common Admission Test)
3. **Merit List**
4. **Counseling**
5. **Document Verification**
6. **Fee Payment**
